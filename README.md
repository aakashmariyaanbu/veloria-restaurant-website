#  Veloria – Fine Dining Restaurant Website

Veloria is a modern multi-page restaurant website built using pure HTML, CSS, and JavaScript.

## Features
- Elegant animated hero section
- Multi-page navigation (Home, Story, Menu, Events, Contact, Reservation)
- Cuisine filter system
- Reservation form with confirmation
- Responsive design
- Modern UI with smooth animations

## Built With
- HTML5
- CSS3
- JavaScript

## Project Structure
- index.html (Main website file)

## How to Run
1. Download the project
2. Open `index.html` in your browser
